{{< products/product-feature-blocks-col
featurecolheading="Hoe OneNote naar PDF te converteren in .NET"
>}}
<p>PDF is het standaardformaat voor het delen van bestanden voor verschillende platforms en apparaten. Door elk document naar PDF te exporteren, kunt u het gemakkelijk delen, openen en weergeven. Aspose.Note voor .NET API is een geweldig OneNote-alternatief en ondersteunt het <a href="https://blog.aspose.com/note/convert-onenote-to-pdf-in-csharp/">converteren van OneNote naar PDF</a>-documenten. De volgende stappen en voorbeeldcode helpen u bij het converteren van OneNote naar PDF. Opgemerkt moet worden dat dit voorbeeld het converteren van bestaande OneNote-documenten benadrukt en niet een volledig nieuw .ONE-bestand maakt voordat het wordt geconverteerd:</p>
<ul>
   <li>Laad een OneNote-bestand met behulp van de klasse <a href="https://reference.aspose.com/note/net/aspose.note/document">Document</a>.</a></li>
   <li>Sla het bestand op als PDF met behulp van de methode <a href="https://reference.aspose.com/note/net/aspose.note.document/save/methods/4">Opslaan</a>. Het neemt het output (PDF) bestandspad en <strong>SaveFormat</strong> als argumenten.</li>
</ul>
{{< products/product-feature-blocks-snip
snipclass="greysnipp"
copyID="snipp2"
>}}
// This code example demonstrates how to convert an existing OneNote to PDF.
// Load the document into Aspose.Note.
Document oneFile = new Document("Sample1.one");
// Save OneNote to PDF
oneFile.Save("Sample1_out.pdf", SaveFormat.Pdf);
{{< /products/product-feature-blocks-snip >}}
{{< /products/product-feature-blocks-col >}}
{{< products/product-feature-blocks-col
featurecolheading="Converteer OneNote naar rasterafbeeldingen in .NET"
>}}
<p>De superieure functieset van de .NET OneNote-automatiserings-API omvat het converteren van OneNote naar JPEG, PNG, BMP en <a href="https://blog.aspose.com/note/convert-onenote-one-files-to-jpeg-png-tiff-image-csharp/">andere rasterafbeeldingen</a>. Converteren naar afbeeldingen is een gemakkelijke manier om documentinformatie te delen en op te slaan. Rasterafbeeldingsindelingen worden over het algemeen ondersteund door alle belangrijke besturingssystemen, mobiele platforms en browsertoepassingen, dus het exporteren van bestanden naar afbeeldingen is ook handig bij samenwerking en opslag. De volgende stappen en informatie laten zien hoe u OneNote naar JPG converteert:</p>
<ul>
   <li>Laad het ingevoerde OneNote-bestand (.ONE) met de klasse <a href="https://reference.aspose.com/note/net/aspose.note/document">Document</a>.</li>
   <li>Sla het op als JPEG-afbeelding met behulp van de methode <a href="https://reference.aspose.com/note/net/aspose.note/document/methods/save/index">Opslaan</a>. Het neemt het uitvoer-JPEG-bestandspad en <strong>SaveFormat</strong> als methodeargumenten.</li>
</ul>
{{< products/product-feature-blocks-snip
snipclass="greysnipp"
copyID="snipp3"
>}}
// Load the input Note (.one) file using the Document class.
Document oneFile = new Document("Aspose.one");
// Save output JPG image.
oneFile.Save("JpegImage_out.jpg", SaveFormat.Jpeg);
