<p>PDF-i Word-ə çevirmə funksionallığına inteqrasiya etməklə mövcud .NET PDF çevirmə proqramlarınızı təkmilləşdirin. .NET API üçün Aspose.PDF, .NET sənəd çevirmə proqramlarınız daxilində PDF-i DOC-a və PDF-ni DOCX-ə çevirməyə imkan verir. Zəhmət olmasa, .NET-də PDF-i Word-ə çevirməkdə kömək etmək üçün aşağıdakı təlimatlara və kod parçasına baxın:</p>
<ul>
   <li>Mənbə PDF sənədi ilə <a href="https://reference.aspose.com/pdf/net/aspose.pdf/document/">Document</a> obyektinin nümunəsini yaradın.</li>
   <li><strong>Document.Save()</strong> metoduna zəng edərək onu <strong>SaveFormat.Doc</strong> formatında saxlayın.</li>
</ul>

public static void ConvertPDFtoWord()
{
    // Open the source PDF document
    Document pdfDocument = new Document(_dataDir + "PDFToDOC.pdf");
    // Save the file into MS document format
    pdfDocument.Save(_dataDir + "PDFToDOC_out.doc", SaveFormat.Doc);
}
