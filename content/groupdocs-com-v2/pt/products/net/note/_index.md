{{< products/product-feature-blocks-col
featurecolheading="Como converter o OneNote para PDF no .NET"
>}}
<p>PDF é o formato padrão de compartilhamento de arquivos para diferentes plataformas e dispositivos. A exportação de qualquer documento para PDF torna possível compartilhá-lo, acessá-lo e renderizá-lo de maneira conveniente. Aspose.Note para .NET API é uma ótima alternativa do OneNote e suporta <a href="https://blog.aspose.com/note/convert-onenote-to-pdf-in-csharp/">convertendo OneNote para PDF</a> documentos. As etapas a seguir e o código de exemplo ajudarão você a converter o OneNote em PDF. Deve-se observar que este exemplo destaca a conversão de documentos existentes do OneNote e não a criação de um arquivo .ONE do zero antes de convertê-lo:</p>
<ul>
   <li>Carregue um arquivo do OneNote usando a classe <a href="https://reference.aspose.com/note/net/aspose.note/document">Document</a>.</a></li>
   <li>Salve o arquivo como PDF usando o método <a href="https://reference.aspose.com/note/net/aspose.note.document/save/methods/4">Salvar</a>. Ele usa o caminho do arquivo de saída (PDF) e <strong>SaveFormat</strong> como argumentos.</li>
</ul>
{{< products/product-feature-blocks-snip
snipclass="greysnipp"
copyID="snipp2"
>}}
// This code example demonstrates how to convert an existing OneNote to PDF.
// Load the document into Aspose.Note.
Document oneFile = new Document("Sample1.one");
// Save OneNote to PDF
oneFile.Save("Sample1_out.pdf", SaveFormat.Pdf);

{{< products/product-feature-blocks-col
featurecolheading="Converter OneNote em imagens raster em .NET"
>}}
<p>O conjunto de recursos superiores da API de automação .NET OneNote inclui a conversão do OneNote para JPEG, PNG, BMP e <a href="https://blog.aspose.com/note/convert-onenote-one-files-to-jpeg-png-tiff-image-csharp/">outras imagens raster</a>. Converter em imagens é uma maneira fácil de compartilhar e armazenar informações de documentos. Os formatos de imagem raster geralmente são suportados em todos os principais sistemas operacionais, plataformas móveis e aplicativos de navegador, portanto, exportar arquivos para imagens também é útil em colaboração e armazenamento. As etapas e informações a seguir mostram como converter o OneNote em JPG:</p>
<ul>
   <li>Carregue o arquivo de entrada do OneNote (.ONE) com a classe <a href="https://reference.aspose.com/note/net/aspose.note/document">Document</a>.</li>
   <li>Salve-o como imagem JPEG usando o método <a href="https://reference.aspose.com/note/net/aspose.note/document/methods/save/index">Salvar</a>. Ele usa o caminho do arquivo JPEG de saída e <strong>SaveFormat</strong> como argumentos do método.</li>
</ul>
{{< products/product-feature-blocks-snip
snipclass="greysnipp"
copyID="snipp3"
>}}
// Load the input Note (.one) file using the Document class.
Document oneFile = new Document("Aspose.one");
// Save output JPG image.
oneFile.Save("JpegImage_out.jpg", SaveFormat.Jpeg);
