featurecolheading="Muunna PowerPoint PDF-muotoon .NET:ssä"
>}}
<p>Suorita virheettömiä PowerPoint-esityksiä PDF-muunnoksiksi C# .NET:ssä käyttämällä Aspose.Slides for .NET API:ta ja määritä samalla tuloksena olevan PDF-asiakirjan erilaisia ominaisuuksia prosessissa. Katso seuraavat vaiheet ja esimerkkikoodaus PowerPoint PPT:n muuntamiseksi PDF-muotoon .NET-alustalla:</p>
<ul>
   <li>Luo <a href="https://apireference.aspose.com/slides/net/aspose.slides/presentation">Presentation</a>-luokkaobjektin esiintymä ladataksesi lähdeesitystiedoston.</li>
   <li>Aseta haluamasi kuvaasetukset alustamalla <a href="https://apireference.aspose.com/slides/net/aspose.slides.export/pdfoptions/">PdfOptions</a>-luokkaobjekti.</li>
   <li>Aseta haluamasi PDF-asetukset tuloksena olevalle PDF-tiedostolle.</li>
   <li>Muunna esitys PDF-muotoon Tallenna-menetelmällä.</li>
</ul>
{{< products/product-feature-blocks-snip
snipclass="greysnipp"
copyID="snipp2"
>}}
// Create an instance of the Presentation class object to load the PPT file
 Presentation presPPT = new Presentation("TestPresentation.ppt");
// Create the PdfOptions class object
PdfOptions exportPdfOptions = new PdfOptions();
// Set the JPEG quality for images inside the presentation
exportPdfOptions.JpegQuality = 90;
// Set the metafiles behavior
exportPdfOptions.SaveMetafilesAsPng = true;
// Apply the text compression level
exportPdfOptions.TextCompression = PdfTextCompression.Flate;
// Set the output PDF standard
exportPdfOptions.Compliance = PdfCompliance.Pdf15;
// Convert and Save the Presentation to PDF
presPPT.Save("PPT-to-PDF.pdf", SaveFormat.Pdf, exportPdfOptions);

{{< products/product-feature-blocks-col
featurecolheading="Yhdistä esitykset .NET:ssä"
>}}
<p>Esitysten yhdistäminen yhtenäiseksi PowerPoint-asiakirjaksi on Aspose.Slides for .NET API:n näkyvä ominaisuus. Tarkista alla olevat vaiheet ja tiedot yhdistääksesi nopeasti ja helposti useita PowerPoint PPTX -esityksiä asiakirjojen yhdistämissovelluksissasi .NET-esitysten API:n avulla:</p>
<ul>
   <li>Luo <a href="https://apireference.aspose.com/slides/net/aspose.slides/presentation">Presentation</a>-luokan esiintymä ladataksesi lähdeesitys.</li>
   <li>Alusta erilliset Presentation-luokan objektit vaadituille kohdeesityksille.</li>
   <li>Iteroi silmukassa kaikki kohdeesitysten <a href="https://apireference.aspose.com/slides/net/aspose.slides/islide">diat</a>.</li>
   <li>Kutsu lähdeesityksen diakokoelman AddClone()-menetelmää, jossa muita esityksiä yhdistetään.</li>
   <li>Tallenna tuloksena oleva esitys, jossa on kaikki kohdeesitysten diat.</li>
</ul>
{{< products/product-feature-blocks-snip
snipclass="greysnipp"
copyID="snipp3"
>}}
// Load the source presentation
Presentation prest1 = new Presentation("prest1.pptx");
// Initialize separate Presentation class objects for the target presentations
Presentation prest2 = new Presentation("prest2.pptx");
Presentation prest3 = new Presentation("prest3.pptx");
// Loop through all target presentations slides
foreach (ISlide slide in prest2.Slides)
    prest1.Slides.AddClone(slide);
 foreach (ISlide slide in prest3.Slides)
    prest1.Slides.AddClone(slide);
// Save the resultant presentation with all the slides from the target presentations
prest1.Save("combined.pptx", SaveFormat.Pptx);
