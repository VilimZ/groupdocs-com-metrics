<p>Déantar aithint optúil carachtar do dhoiciméid PDF scanta le cabhair ó Aspose.OCR do .NET. Tá sé ar cheann de na gnéithe is mó a thaitin leis an .NET PDF document OCR API agus cuireann sé na huirlisí riachtanacha go léir ar fáil d'aon duine atá ag obair leis an API chun téacs ó dhoiciméid PDF a aithint, a bhaint agus a thiontú go formáidí éagsúla. Breathnaímid ar na céimeanna is gá chun OCR a dhéanamh ar dhoiciméid PDF agus an téacs asbhainte a thiontú ó PDF go formáid Word:</p>
<ul>
   <li>Cuir an rang <strong>AsposeOcr</strong> ar bun.</a></li>
   <li>Cruthaigh réad den rang <strong>Socruithe Aitheantas Doiciméid</strong>.</li>
   <li>Sonraigh an teanga OCR.</li>
   <li>Glaoigh ar an modh <strong>RecognizePdf()</strong> chun <strong>Torthaí Aitheantas</strong> a fháil. Cuir cosán comhaid agus oibiacht <strong>DocumentRecognitionSettings</strong> leis mar argóintí.</li>
   <li>Sábháil an téacs leis an modh <strong>SaveMultipageDocument()</strong>. Cuir cosán an chomhaid aschuir, an <strong>SaveFormat</strong>, agus <strong>RecognitionResult</strong> leis mar argóintí</li>
</ul>
{{< products/product-feature-blocks-snip
snipclass="greysnipp"
copyID="snipp2"
>}}
// This code example demonstrates how to OCR PDF documents and save the recognized text as DOCX.
// Initialize the PCR engine
AsposeOcr recognitionEngine = new AsposeOcr();
// Initialize recognition settings
DocumentRecognitionSettings recognitionSettings = new DocumentRecognitionSettings();
// Specify language for OCR. Multi-language by default
recognitionSettings.Language = Language.Eng;
// Recognize text from PDF
List<RecognitionResult> results = recognitionEngine.RecognizePdf("C:\\MyFiles\\test.pdf", recognitionSettings);
// Save the recognized text as Word (DOCX) file
AsposeOcr.SaveMultipageDocument("C:\\Files\\OCR_result.docx", SaveFormat.Docx, results);

<p>Ligeann an API .NET OCR ardfheidhmíochta d’fhorbróirí téacs inchuardaithe a bhaint as íomhánna scanta agus grianghraif de bhformáidí éagsúla íomhá a úsáidtear go coitianta ar nós GIF, PNG, JPG, BMP, agus TIFF. Má tá tú ag iarraidh cumais OCR a chomhtháthú isteach i d'aipeanna .NET, d'fhéadfadh an ghné tiontaithe íomhá go téacs iad a neartú. Anseo, déanfaimid athbhreithniú ar na céimeanna atá riachtanach chun JPG a thiontú go PDF go héifeachtach trí úsáid a bhaint as an API OCR láidir do .NET:</p>
<ul>
   <li>Socraigh cosán comhaid íomhá ionchuir chun téacs a aithint.</li>
   <li>Cruthaigh sampla ranga <strong>AsposeOcr</strong></li>
   <li>Úsáid an modh <strong>AithnighÍomhá</strong> chun téacs a aithint i do chomhad íomhá roghnaithe.</li>
   <li>Sábháil an comhad próiseáilte leis an modh <strong>Sábháil</strong>, agus cuir ainm an chomhaid aschuir agus an <strong>SaveFormat</strong> leis mar argóintí.</li>
   <li>Cinntigh go socrófar an comhad mar thoradh ar dhoiciméad PDF inchuardaithe ag úsáid <strong>SaveFormat</strong>.</li>
</ul>
