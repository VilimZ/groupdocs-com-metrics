A .NET diagram API segítségével a semmiből hozhat létre Visio dokumentumokat az alábbi lépések és információk segítségével:</p>
<ul>
   <li>Ellenőrizze, hogy létezik-e a fájl mentésére szolgáló könyvtár.</li>
   <li>Hozzon létre egy könyvtárat, ha nem létezik.</li>
   <li>Inicializálja a <a href="https://reference.aspose.com/diagram/net/aspose.diagram/diagram">Diagram</a> osztály objektumát.</li>
   <li>Mentse el a diagramot VSDX-ként a <a href="https://reference.aspose.com/diagram/net/aspose.diagram.diagram/save/methods/2">Diagram.Save(String fileName, SaveFileFormat.VSDX)< használatával. /a> módszer.</li>
</ul>

// The path to the documents directory.
string dataDir = RunExamples.GetDataDir_Diagrams();
// Create a directory if it is not already present.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
// Initialize a new Visio
Diagram diagram = new Diagram();

// Save in the VSDX format
diagram.Save(dataDir, SaveFileFormat.VSDX);

<p>A fájlok PDF formátumba konvertálása megkönnyíti a megosztást és a hozzáférést több eszközön. Az Aspose.Diagram for .NET API lehetővé teszi a Visio diagramok PDF formátumba konvertálását a C# .NET-alkalmazásokon belül. Programozottan konvertálhat VSD-t vagy VSDX-et PDF-be, kiváló konverziós minőséggel. A következő lépések és információk segítenek ennek elérésében:</p>
<ul>
   <li>A konvertálandó bemeneti Visio-fájl betöltéséhez inicializálja a <a href="https://apireference.aspose.com/diagram/net/aspose.diagram">Diagram</a> osztályobjektumot.</li>
   <li>Példányosítsa a <a href="https://reference.aspose.com/diagram/net/aspose.diagram.saving/pdfsaveoptions/">PdfSaveOptions</a> osztályobjektumot.</li>
   <li>Állítsa be a fájl tulajdonságait, például az oldalszámot és az oldalindexet.</li>
   <li>Mentse el a kimeneti fájlt PDF-ként.</li>
</ul>
{{< products/product-feature-blocks-snip
snipclass="greysnipp"
copyID="snipp3"
>}}
// Load Visio diagram
Aspose.Diagram.Diagram diagram = new Aspose.Diagram.Diagram("Sample.vsd");
// Declare PdfSaveOptions object
Aspose.Diagram.Saving.PdfSaveOptions saveOptions = new Aspose.Diagram.Saving.PdfSaveOptions();
// Number of pages to render
saveOptions.PageCount = 2;
// Set first page index
saveOptions.PageIndex = 1;
// Save Visio diagram to PDF
diagram.Save("PDF_out.pdf", saveOptions);
Console.WriteLine("Done");
